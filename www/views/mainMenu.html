This is a player for Lone Wolf game books 1 to 26.

Only latest versions of Firefox and Chromium based browsers are supported. Other browsers and versions are not supported (they may work... or not).

Keep in mind that if you clean your browser history, the current game state will be lost. You can save the current game to a file at Settings > Save game.

Have fun!
